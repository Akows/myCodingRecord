자바스크립트는 본래 HTML/CSS와 함께 프론트엔드 내에서 사용되는 수준에 불과한 언어였다.

그런데 자바스크립트의 동작은 웹 브라우져에서 개발한 자바스크립트 엔진 내에서만 가능했다.
헌데 웹 개발과 관련되어 이미 널리 사용되고 있는 자바스크립트를 웹 브라우저 밖에서도 사용하고자 하는 욕구가 탄생한다.

이를 위해 개발된 것이 Node.js. 자바스크립트가 독자적으로 동작하도록 환경을 조성해주는 역할을 한다.
브라우저에 종속되지 않고 독립성을 가짐으로써 자바스크립트의 영역이 웹 프론트를 벗어나 더 넒은 장소를 가지게 된 것.

대표적인 예시가 자바스크립트를 이용한 '웹 서버 개발'이 가능해진 점이다.

또한 Node를 기반으로 만들어진 것 중 하나가 React
브라우저에서 동작하는, 복잡하고 어려운 자바스크립트 파일을 '자동으로' 만들어내는 웹 어플리케이션이다.

NPM : Node.js의 기본 패키지 관리 도구.
패키지? 다른 누군가가 미리 만들어 둔 기능 등의 집합체.
